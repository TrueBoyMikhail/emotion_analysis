# Анализ мультимодальных данных в распознавании эмоций

> Репозиторий содержит код, архитектуры и результаты экспериментов курсовой работы по разработке и сравнению моделей машинного обучения для распознавания эмоций по аудиосигналам и изображениям лиц.

## О проекте

Целью данной работы является исследование методов анализа мультимодальных данных и разработка подхода к распознаванию эмоций на основе технологий машинного и глубокого обучения. В рамках проекта реализованы и сравнены несколько архитектур нейронных сетей.

**Ключевые задачи, решенные в проекте:**
- Сбор и объединение 5 различных датасетов (аудио + изображения).
- Предварительная обработка сигналов (извлечение MFCC, ZCR, RMS) и аугментация данных.
- Разработка 1D/2D CNN, архитектуры с механизмом Attention (BiLSTM) и применение Transfer Learning (Wav2Vec2).
- Сравнительный анализ метрик качества (Accuracy, F1-score, Confusion Matrix).

## Технологический стек

- **Язык:** Python 3.9+
- **Deep Learning:** TensorFlow, Keras, Hugging Face `transformers`
- **Обработка данных:** Librosa, NumPy, Pandas, OpenCV
- **Анализ и визуализация:** Scikit-learn, Matplotlib, Seaborn
- **Среда:** Jupyter Notebook / Google Colab / Kaggle

## Используемые наборы данных

Проект использует данные из следующих открытых источников:

| Модальность | Датасет | Описание | Ссылка |
| :--- | :--- | :--- | :--- |

| **Аудио** | **SAVEE** | 4 носителя языка, 7 эмоций | [Kaggle](https://www.kaggle.com/datasets/ejlok1/surrey-audiovisual-expressed-emotion-savee) |
